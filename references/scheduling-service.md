# 飞书约面服务

## 状态

**待接入** — 飞书 bot 配置完成后启用。

---

## 目标

候选人已确认面试时间后，通过飞书 bot 创建日程，并将日程链接发回候选人。

---

## 进入条件

同时满足以下条件才能进入：

1. 用户明确同意该候选人进入约面
2. 候选人已在聊天中确认具体可面试时间

---

## 飞书 bot 接入（首次使用时引导）

第一次真正要创建日程时，引导用户完成以下步骤：

1. 在飞书开放平台创建应用，获取 `App ID` 和 `App Secret`
2. 将凭证提供给 skill
3. 在飞书 bot 聊天中完成日历授权
4. 授权完成后，skill 通过 bot API 创建日程

skill 不主动打开飞书开放平台，不替用户点击后台。

---

## 日程创建流程

1. 确认面试时间、时长、参会人
2. 调用飞书 bot API 创建日程
3. 获取日程链接
4. 通过 BOSS 站内消息将链接发给候选人（走 chat-service 发送流程）
5. 更新候选人状态为 `interview_scheduled`

---

## 默认参会人

默认只加 HR 本人。
如需加其他固定参会人，在首次配置时确认。
